## 📖 Descripción

**SpeakUp Pro** es una aplicación web educativa para aprender inglés conversacional basada en la ciencia del aprendizaje. Combina las técnicas más efectivas comprobadas por la neurociencia.

## ✨ Características

- 📚 **Niveles MCER** (A1-C2) con cuestionarios.
- 🔁 **Sistema SRS tipo Anki** (algoritmo SM-2) para repaso espaciado.
- 🍅 **Pomodoro personalizable** con tiempos basados en neurociencia.
- 🧠 **Sección de Neurociencia** con métodos científicos.
- 🎯 **Modo Enfoque** para máxima concentración.
- 🎁 **Incentivos sorpresa** con confetti.
- 🏆 **Ranking** de estudiantes.
- 🎖️ **Sistema de logros** y medallas.
- 📺 **YouGlish** integrado (pronunciación de nativos).
- 🔊 **Síntesis de voz** (TTS).
- 💾 **Progreso guardado** automáticamente (`localStorage`).

## 🚀 Tecnologías

- HTML5
- CSS3 (variables, grid, flexbox, animaciones)
- JavaScript Vanilla (sin frameworks)
- Web Speech API
- `localStorage`
- YouGlish Widget API
